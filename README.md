# Bereshit

Screen brightness and gamma control app made with PySide2 

### Dependencies

* Python3
* PySide2
* Xrandr

### Installing & Executing program

```
$ git clone https://github.com/Almog101/Bereshit.git

$ cd Bereshit

$ pip3 install -r requirements

$ python3 main.py
```
